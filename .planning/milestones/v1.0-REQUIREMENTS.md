**Archived:** 2026-02-24

## Functional Requirements

- [x] **FR-01:** Users authenticate via Google SSO; only @cast.ai domain allowed
- [x] **FR-02:** System accesses Google Drive folder via service account (no user OAuth for Drive)
- [x] **FR-03:** Parse Gong transcript format — extract call ID, date, title, participants (name + email), speaker turns (ID, timestamp, text, topic tags)
- [x] **FR-04:** Match authenticated user to call participants via email for access control
- [x] **FR-05:** Analyze transcripts with Claude across 8 dimensions: summary, overall sentiment, sentiment trajectory, objections, objection handling, action items, topics, talk ratio, competitor mentions
- [x] **FR-06:** Map opaque speaker IDs to participant names using conversational cues
- [x] **FR-07:** Store users, calls, and analyses in Firestore with versioned analysis history
- [x] **FR-08:** Dashboard shows user's calls with sentiment and topic preview
- [x] **FR-09:** Call detail page displays full analysis with all 8 dimensions
- [x] **FR-10:** Export analysis as Markdown report
- [x] **FR-11:** Analysis runs in background; frontend polls for completion
- [x] **FR-12:** Re-analysis creates new version (does not overwrite)

## Non-Functional Requirements

- [x] **NFR-01:** TypeScript strict mode across entire codebase
- [x] **NFR-02:** Structured JSON logging for Cloud Run compatibility
- [x] **NFR-03:** Custom error hierarchy with operational vs programmer error distinction
- [x] **NFR-04:** Retry with exponential backoff on Google Drive API (429/5xx)
- [x] **NFR-05:** Retry with exponential backoff on Claude API (429/529/5xx)
- [x] **NFR-06:** 90-second timeout on Claude analysis calls
- [x] **NFR-07:** 3-minute timeout on background analysis jobs
- [x] **NFR-08:** Startup validation of all required environment variables
- [x] **NFR-09:** JWT failure logging with specific reason (expired, malformed, bad signature)
- [x] **NFR-10:** React ErrorBoundary prevents white-screen crashes
- [x] **NFR-11:** Frontend polling capped at 90 iterations (3 min) with cleanup
- [x] **NFR-12:** Global Hono error handler with request ID correlation
- [x] **NFR-13:** Deployment-ready Dockerfile and Cloud Run deploy script
